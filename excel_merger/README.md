# Excel Merger

This script combines multiple Excel files into a single file.

## Features
- Merges multiple .xlsx files
- Automatically reads all files in a folder
- Outputs a single combined Excel file

## Usage
1. Place your Excel files inside the "input_excels" folder
2. Run the script:
   python main.py
3. Get the merged file: merged_output.xlsx

## Requirements
- Python
- pandas
